Política de privacidad

Aplicación: ForoPlus (cliente no oficial para el foro forocoches.com)
Última actualización: 28 de junio de 2026

Esta aplicación es un cliente no oficial e independiente para el foro forocoches.com. No está afiliada, asociada, autorizada ni respaldada por Forocoches ni por los titulares de su marca. Su función es mostrar el foro dentro de una WebView y añadir mejoras de uso (filtros, avisos de mensajes privados).

1. Resumen

La aplicación no recopila, no almacena en servidores y no comparte datos personales con el desarrollador ni con terceros. No incluye analítica, ni publicidad propia, ni rastreadores. Toda la información se procesa en tu dispositivo.

2. Datos que se manejan en el dispositivo

  • Sesión e inicio de sesión: cuando inicias sesión en Forocoches, las cookies de sesión las gestiona la WebView y se guardan únicamente en tu dispositivo. La aplicación no las envía a ningún servidor del desarrollador.
  • Avisos de mensajes privados y notificaciones: para mostrarte un aviso, la aplicación consulta forocoches.com con tu propia sesión y lee el número de mensajes y una vista previa del último. Ese contenido se usa para crear la notificación local y no se almacena ni se transmite a terceros.
  • Preferencias de filtrado (lista de usuarios ignorados y palabras clave): se guardan localmente. Una lista de configuración de filtros se descarga desde un repositorio público alojado en GitHub; esa petición no envía datos personales tuyos.

3. Conexiones de red

La aplicación se conecta a:

  • forocoches.com — para mostrar el foro y comprobar tus mensajes/avisos. El uso del foro se rige por la política de privacidad del propio Forocoches.
  • GitHub — para descargar el fichero público de configuración de filtros.

4. Permisos

  • INTERNET — cargar el foro y comprobar avisos.
  • POST_NOTIFICATIONS — mostrar avisos de mensajes privados y notificaciones.
  • FOREGROUND_SERVICE / FOREGROUND_SERVICE_DATA_SYNC — comprobar periódicamente si tienes mensajes nuevos.
  • RECEIVE_BOOT_COMPLETED — reanudar la comprobación de avisos tras reiniciar.
  • REQUEST_IGNORE_BATTERY_OPTIMIZATIONS — evitar que el sistema detenga los avisos en segundo plano (opcional; lo concedes tú).

5. Datos de menores

La aplicación no está dirigida a menores de edad y no recopila intencionadamente datos de menores.

6. Cambios

Si esta política cambia, se actualizará esta misma página con una nueva fecha de revisión.
